Input Groups

An input element can be grouped with its label and an add on, checkbox ,or button element to create a handy standalone component or for form building.

Basic Input Group with Addon

The standard input group contains a label and addon. The addon can be placed on the left, right or both sides of the input.

@
github.com
$ .00 
              
                <div class="input">
                  <label for="" class="input__label">Left addon</label>
                  <div class="input__container input--md">
                    <span class="input__addOn input--addOnLeft">@</span>
                    <input type="text" class="input__control control--rtRadius"  placeholder="Avatar Name">
                  </div>
                </div>

                <div class="input">
                  <label for="" class="input__label">Right addon</label>
                  <div class="input__container input--md">
                    <input type="text" class="input__control control--lftRadius" placeholder="username">
                    <span class="input__addOn input--addOnRight">github.com</span>
                  </div>
                </div>

                <div class="input">
                  <label for="" class="input__label">Both-sided addon</label>
                  <div class="input__container input--md">

                    <input type="text" class="input__control" placeholder="">
                    <span class="input__addOn input--addOnRight">.00</span>
                  </div>
                </div>

With Buttons

The addon can also be replaced with a button for immediate action.

              
                <div class="input">
                  <label for="" class="input__label"></em></label>
                  <div class="input__container input--lg">
                    <button class="input__button">Send!</button>
                    <input type="email" class="input__control"  placeholder="Email">
                  </div>
                </div>
                <div class="input">

                  </div>
                </div>
