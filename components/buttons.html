Button Sizes

Buttons come in four height sizes as well as four width sizes. Extra small, small, medium, and large. Medium is the default size if no other is specified. Add any combination of the width and height classes for the desired size.

For extra small they are .btn-xs-w and .btn-xs-h. For small they are .btn-sm-w and .btn-sm-h. For large they are .btn-lg-w and .btn-lg-h.

The button width can also be dictated by the text with appropriate padding. In this case only a .btn-*-h needs to be specified.

Buttons can also be the entire width of their parent container by adding the .btn-full-w. If desired, the button width could also be determined by the parent container having the appropriate grid classes for positioning and dimensioning.

Button Weght

Buttons can have a weight, or "presence", of either raised, flat, or sunken. The -raised button is the default style with a slight box-shadow. The raised with border button is a raised button with a border and has the class btn-raisedBorder. A -flat button has no border or box-shadow and takes the class btn-flat. The sunken button has a border with a inset box shadow and takes the class btn-sunken.

Rotate 45 button

All button types need to be wrapped in a .rotate__container div and each button type except the actual button tag, will get the btn-rotate45 class in a span surrounding the button.

The clases are .btn-rotate45-xs, .btn-rotate45-sm, .btn-rotate45-md, and .btn-rotate45-lg.

The .btn-rotate45-xs can also be used as a link, for example in site navigation. Simply embed the .rotate__container in the navigation <li>.
